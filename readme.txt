"Mt8 Import Comments Adjuster" is a plugin to adjust comments of importing export data of the FC2 blog.

== Description ==

Mt8 Import Comments Adjusterは、FC2ブログのエクスポートデータをインポートする際のコメント内容を調整するプラグインです。
"管理者にだけ表示を許可する"になっているコメントは"承認待ち"にします。
「Movable Type and TypePad Importer」を先にインストールしておいて下さい。

== Installation ==

1. Upload `plugin-name.zip` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
